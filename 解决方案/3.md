# 标签管理

## 1 创建`tag`

```shell
# 默认基于HEAD新建标签
$ git tag <tagname>

# 基于commitid新建标签
$ git tag <tagname> <commitid>

# 新建带备注的标签，-a参数来创建一个带备注的tag，备注信息由-m指定
$ git tag -a <tagname> -m "描述信息" 
```

## 2 `tag`推送到远程

```shell
# 推送本地标签到远程
$ git push origin <tagname>

# 推送全部未推送过的本地标签
$ git push origin --tags
```

## 3 删除`tag`

```shell
# 删除本地标签
$ git tag -d <tagname>

# 删除远程标签
$ git push origin :refs/tags/<tagname>
```

## 4 查看tag

```shell
# 查看所有标签
$ git tag

# 查看远程仓库的标签
$ git ls-remote --tags
```
